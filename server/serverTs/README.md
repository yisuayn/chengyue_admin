## Install

***本地测试环境***
```bash

npm install 
sh run.sh test
```

***本地开发环境***
```bash

npm install 
sh run.sh dev
```

### 启动配置
可自行调整进程数量及其他
```bash
export PORT=3000
export TZ=Asia/Shanghai
# 运行环境
export SERVER_ENV=test
export HOSTNAME=paramecium
export VERSION=v1.1.0
export NUM_CPUS=1
```
